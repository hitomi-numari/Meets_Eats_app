# Meets Eats
## Concept
ご飯を奢りたい人と奢られたい人が食事を共にするマッチングアプリ
## Overview
食事をご馳走する代わりに誰かと会話を楽しみたい人向けのアプリケーションです。
会話したいテーマ別に食事相手を探せます。
# バージョン
Ruby 2.6.5
Rails 5.2.4
